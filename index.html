O que é?

O webBudget é um sistema grátis e de código aberto para controle financeiro pessoal ou de pequenas empresas.

Com ele você poderá fazer organização de suas receitas e despesas dentro de um período de tempo, realizar a classificação de sua movimentação financeira através de centros de custos e tipos de movimentação além de poder acompanhar tudo isso em tempo real na dashboard do sistema

Movimentação

Controle de receitas e despesas fixas ou variáveis dentro de períodos pré estabelecidos

Classificação

Crie classes de receitas ou despesas e vincule aos centros de custo para auxiliar na classificação de sua movimentação

Cartões

Cadastre e controle a movimentação realizada em seus cartões de crédito ou débito

Carteiras

Utilize para controlar os saldos de suas contas bancárias ou de sua carteira pessoal

Controle de acessos

Crie grupos de acesso customizados, libere acesso somente as funcionalidades que quiser para as pessoas que precisam

Dashboard

Veja todo o andamento de suas finanças no período aberto pela dashboard do sistema em tempo real com gráficos

Tecnologias

  • Wildfly 13
  • Deltaspike Data + Hibernate
  • Arquitetura baseada nos frameworks do JavaEE versão 7
  • Primefaces + Omnifaces + AdminLTE + Bootstrap 3
  • Homologado para PostgreSQL 9+ ou MySQL 8/MariaDB 10+
  • Segurança provida pelo ShiroEE, uma extensão do Apache Shiro

Utilização

Existem três maneiras documentadas para utilizar o webBudget: utilizando um container docker, em uma infraestrutura que você poderá montar ou em uma plataforma PaaS como o Jelastic.

Docker

A maneira mais simples de executar o projeto, para isso siga os passos abaixo:

  • Instale o docker seguindo os passos mostrados aqui para linux ou aqui para windows
  • Com o docker instalado execute o comando abaixo em um terminal (no linux) ou no powerShell (no windows):


docker run --name webbudget -itd -p 8443:8443 arthurgregorio/web-budget-docker

Feito isso, se nenhum erro ocorrer, você poderá abrir o browser e o sistema estará disponível no endereço https://localhost:8443.
